# SQL & BigQuery Practice

This repository contains SQL and BigQuery practice queries created for
Data Engineering interview preparation.

## Topics Covered
- Basic SELECT and filtering
- Aggregations and GROUP BY
- Joins (INNER, LEFT)
- Subqueries and CTEs
- Window functions (ROW_NUMBER, RANK, DENSE_RANK, LAG, LEAD)
- Duplicate handling
- BigQuery-specific features
- MERGE (Upsert) logic
- Stored procedures

## Purpose
The goal of this repository is to practice and demonstrate
real-world SQL patterns used in analytics and data engineering projects.

## Tools
- Google BigQuery
- Standard SQL
- Git & GitHub
- VS Code
